# Brief de producto

## Visión

Crear una pequeña web app que acompañe a la pareja del usuario durante una cita
con varias paradas. La app funciona como guía del recorrido: muestra qué hacer
en cada momento y revela el paso siguiente al completar el actual, sin
reemplazar la experiencia presencial.

## Persona principal

La pareja del creador, probablemente usando el teléfono mientras se desplaza.
Por eso la interfaz debe ser clara con una mano, rápida de cargar y legible en
exteriores.

## Promesa

“No necesitás conocer el destino; la app te acompaña a descubrirlo, paso a paso.”

## Alcance confirmado para la base

- Aplicación React estática.
- Portada romántica con un llamado a comenzar.
- Workflow secuencial con estados disponible, bloqueado y completado.
- Recorrido real de nueve pasos para un viaje de Colonia a Buenos Aires y vuelta.
- Bloques de sugerencias para desayuno, comida y compras veganas.
- Mapas embebidos y enlaces para abrir cada opción en Google Maps.
- Diseño mobile-first, cálido y personal.
- Preparación para publicación automática en GitHub Pages.
- Documentación suficiente para continuar el diseño en otras sesiones.

## Fuera de alcance por ahora

- Backend, cuentas de usuario o autenticación.
- Geolocalización automática.
- Notificaciones o integración con calendarios.
- Formularios, almacenamiento remoto o analítica.

Estos elementos podrán evaluarse cuando el recorrido y su mecánica estén
definidos.
