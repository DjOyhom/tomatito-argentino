# Principios de experiencia

## Tono

- Íntimo y juguetón, nunca corporativo.
- Español rioplatense: “vos”, “preparate”, “dejate sorprender”.
- Frases cortas que puedan leerse caminando.
- Misterio amable: generar curiosidad sin frustrar.

## Diseño

- Mobile-first; escritorio es una ampliación cuidada, no el caso principal.
- Paleta cálida inspirada en papel, terracota y verde profundo.
- Tipografía editorial para emoción y sans serif para información.
- Un objetivo principal por pantalla.
- Estados bloqueados comprensibles y revelaciones celebratorias.
- Animación sutil, respetando `prefers-reduced-motion`.

## Accesibilidad y uso real

- HTML semántico, foco visible y controles con nombres claros.
- Contraste suficiente y texto legible en pantallas pequeñas.
- No depender solo del color para indicar progreso o bloqueo.
- Evitar interacciones que requieran precisión o conexión constante.
- No exponer direcciones, horarios o datos personales en el repositorio público
  sin consentimiento explícito.
